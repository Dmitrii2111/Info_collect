Отключить доступ?

Анна Смирнова

Активен
person a.smirnova work Оператор

Отключение доступа заблокирует вход в систему. Все записи об инспекциях, отчеты и история действий будут сохранены в соответствии с регламентом хранения данных.

Что изменится:

  • cancel Статус сотрудника изменится на «Отключён»
  • logout Все активные веб и мобильные сессии будут немедленно завершены
  • block Доступ к мобильному приложению InfoCollect будет заблокирован
  • history История выполненных инспекций останется доступна для аудита
sync Синхронизация с базой данных произойдет мгновенно
